## Mengenal java
Java adalah bahasa pemrograman tingkat tinggi yang bersifat objek (Object-Oriented Programming). Salah satu ciri khasnya yang paling terkenal adalah prinsip "Write Once, Run Anywhere" (WORA). Artinya, kode yang kita tulis di satu komputer bisa dijalankan di berbagai perangkat lain tanpa harus diubah ulang, selama perangkat tersebut memiliki Java Virtual Machine (JVM

## Apa itu WORA?
WORA adalah singkatan dari "Write Once, Run Anywhere".

Secara tradisional, jika kamu menulis kode dalam bahasa seperti C atau C++, kode tersebut harus dikompilasi (diubah menjadi bahasa mesin) khusus untuk sistem operasi tertentu. Kode untuk Windows tidak bisa langsung dijalankan di Mac tanpa diubah dan dikompilasi ulang.
Java memecahkan masalah ini dengan cara yang unik:

* Compiler: Saat kamu menulis kode Java (.java), kamu tidak langsung mengubahnya menjadi bahasa mesin komputer. Kamu mengubahnya menjadi format antara yang disebut Bytecode (.class).
* JVM (Java Virtual Machine): Di sinilah keajaibannya. Setiap perangkat (baik itu laptop Windows, server Linux, atau bahkan kulkas pintar) memiliki "penerjemah" bernama JVM.
* Eksekusi: JVM akan membaca Bytecode tersebut dan menerjemahkannya ke bahasa yang dimengerti oleh perangkat keras spesifik tempat ia berada.
* Hasilnya: Kamu cukup menulis dan mengompilasi kode satu kali, dan ia bisa berjalan di mana saja asalkan ada JVM di sana.

## Sejarah singkat java
Java tidak diciptakan secara tidak sengaja. Berikut adalah sekilas bagaimana ia bermula:
* Proyek Hijau (Green Project): Pada awal 1990-an, sebuah tim di Sun Microsystems yang dipimpin oleh James Gosling ingin menciptakan bahasa untuk perangkat elektronik konsumen (seperti remote TV atau mesin cuci).
* Awalnya Bernama Oak: Awalnya bahasa ini dinamakan Oak, terinspirasi dari pohon ek yang terlihat dari jendela kantor Gosling. Namun, karena nama tersebut sudah menjadi merek dagang perusahaan lain, mereka mencari nama baru.
* Inspirasi Kopi: Nama "Java" akhirnya dipilih, yang konon terinspirasi dari kopi Jawa (Java Coffee) yang banyak dikonsumsi oleh tim pengembang saat itu. Itulah mengapa logo Java berbentuk cangkir kopi yang mengepul
